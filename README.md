# authentication
IITJ Software Engineering Project: OAuth/openid based authentication portal

## LintTest
> npm run lintTest

## Lint
> npm run lint

## Build
> npm run build

## Test
#### Without build & coverage
> npm run localTests

#### Without build
> npm run localTestsWithCoverage

#### With build and Coverage
> npm test

## Start Server
> npm start
