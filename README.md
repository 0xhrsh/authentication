# NPM Scripts

 1. **Lint Test**
	```sh
	$ npm run lintTest
	```

2. **Lint**
	```sh
	$ npm run lint
	```

3. **Build**
	```sh
	$ npm run build
	```

4. **Tests**
	```sh 
	# Without build & coverage
	$ npm run localTests

	# Without build
	$ npm run localTestsWithCoverage

	# With build and Coverage
	$ npm test
	```

5.  **Start Dev Server**
	```sh
	$ npm start
	```
